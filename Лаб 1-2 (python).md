### Задание на первую лабораторную работу: 
Написать функцию solve для решения уравнения вида `ax^2 + bx + c`. Функция принимает параметры a, b, c и возвращает единственный ответ:
* Если ответа два, то наименьший из них
* Если ответов бесконечное множество, то -1
* Если ответов нет, то None

Декомпозировать эту функцию на несколько: 
* `discr(a, b, c)` - функция нахождения дискриминанта
* `lin_solve(k, b)` - функция решения линейного уравнения
* `sq_solve(a, b, c)` - функция решения квадратного! уравнения
* `solve(a, b, c)` - функция решения уравнения вида ax^2 + bx + c с лBюбыми коэффициентами a, b, c
Протестировать функции вручную, а затем написать по одному единичному тесту на каждую из них.
