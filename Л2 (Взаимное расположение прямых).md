# Подробное руководство по TDD для определения взаимного расположения прямых

## Постановка задачи

Нужно реализовать функцию, которая определяет взаимное расположение двух прямых, заданных уравнением вида: **ax + by + c = 0**

Возможные случаи:
- **Параллельные** - прямые не пересекаются
- **Совпадающие** - это одна и та же прямая
- **Перпендикулярные** - пересекаются под углом 90°
- **Пересекающиеся** - пересекаются под любым другим углом

## Критерии определения

1. **Параллельные** (11): `a1/a2 = b1/b2 ≠ c1/c2`
2. **Совпадающие** (1): `a1/a2 = b1/b2 = c1/c2`  
3. **Перпендикулярные** (0): `a1*a2 + b1*b2 = 0`
4. **Пересекающиеся** (-1): все остальные случаи

## Важные моменты для реализации

### Проблема деления на ноль
Используйте умножение вместо деления:
```cpp
// Вместо a1/a2 = b1/b2
if (a1 * b2 == a2 * b1) {
    // Прямые параллельны или совпадают
}
```

### Точность вычислений с float (если не проходят, но должны)
Используйте epsilon для сравнения:
```cpp
const float EPS = 1e-6;
if (fabs(a1 * b2 - a2 * b1) < EPS) {
    // Прямые параллельны или совпадают
}
```

### Порядок проверок
Рекомендуемый порядок:
1. Совпадающие (частный случай параллельных)
2. Параллельные
3. Перпендикулярные
4. Пересекающиеся
